Welcome on the Domogik packages repository!

This packages repository is similar as a store or market for smartphones. You will find all the packages for Domogik, a home automation system which is free and open source.

Go and search for packages!

How to use this package repository ?

If you are not a package developper, you will have access to three sections :

  • Packages: this section will list all the validated packages. This is the place where you should look for packages.
  • Submitted packages: this section is mainly for your information. You will the packages submitted by the packages developpers but not yet validated (or refused) by the Domogik team.
  • Dashboard: this section will give you an overview of all the components and packages status. This section can be used to have an overview of the build status of all components and packages. In this section you will also find the list of all the packages in development.

If you are a package developper, with a GitHub account you can login to the root packages repository. When logged in, you will be able to submit some packages and to access the list of all the refused packages. {% if not g.build %}
